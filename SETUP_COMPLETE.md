### 🚀 部署步骤总结

1. **创建GitHub仓库**
   ```bash
   # 在GitHub创建新仓库
   git clone https://github.com/your-username/financial-news.git
   ```

2. **上传项目文件**
   ```bash
   # 复制所有文件到仓库目录
   cp -r 金融热点/* your-repo/
   ```

3. **获取QQ邮箱授权码**
   - 登录QQ邮箱网页版
   - 设置 → 账户 → POP3/SMTP服务
   - 开启服务并获取16位授权码

4. **配置GitHub Secrets**
   - `EMAIL_USERNAME` = 你的QQ邮箱地址
   - `EMAIL_PASSWORD` = 16位授权码

5. **启用工作流**
   - 进入GitHub仓库Actions页面
   - 启用"金融快报邮件发送"工作流

### 🔧 自定义选项

#### 修改接收邮箱
编辑 `config.json`：
```json
{
  "email": {
    "to_address": "your-email@example.com"
  }
}
```

#### 修改发送时间
编辑 `.github/workflows/financial_news_email.yml`：
```yaml
schedule:
  - cron: '0 1 * * *'  # 修改cron表达式
```

#### 常用时间设置
- `0 1 * * *` = 北京时间9:00
- `0 2 * * *` = 北京时间10:00
- `0 0 * * 1-5` = 工作日北京时间8:00
