## 🔄 工作流程

1. **每日9:00触发**: GitHub Actions自动运行
2. **获取真实新闻**: 从配置的RSS源和API获取前一天新闻
3. **内容处理**: 清理、格式化、提取关键信息
4. **生成邮件**: 创建包含真实新闻的邮件内容
5. **发送邮件**: 通过配置的邮箱发送给用户
6. **错误处理**: 如果新闻源不可用，使用备用内容
