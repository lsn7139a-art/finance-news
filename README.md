# 金融资讯分析与邮件生成系统

这是一个自动化的金融资讯收集、分析和邮件发送系统，专为每日金融快报设计。

## 功能特点

- **多来源资讯收集**：A股、美股、加密货币三大市场
- **结构化内容生成**：每条资讯包含标题、摘要、链接、影响分析和投资建议
- **自动化邮件发送**：通过GitHub Action定时发送
- **北京时间9:00发送**：确保及时性

## 资讯配置

- **A股资讯**：4条（同花顺、东方财富等来源）
- **美股资讯**：3条（Bloomberg、Reuters、CNBC等来源）
- **加密货币资讯**：3条（CoinDesk、Cointelegraph等来源）
- **总计**：10条资讯

## 设置步骤

### 1. GitHub仓库设置

1. 创建新的GitHub仓库
2. 上传所有文件到仓库
3. 在仓库设置中添加Secrets：
   - `EMAIL_USERNAME`：您的QQ邮箱地址
   - `EMAIL_PASSWORD`：QQ邮箱授权码（不是密码）

### 2. 获取QQ邮箱授权码

1. 登录QQ邮箱网页版
2. 进入设置 → 账户
3. 找到"POP3/SMTP服务"，点击"开启"
4. 按照提示获取授权码

### 3. 启用GitHub Action

1. 在GitHub仓库中进入Actions页面
2. 启用工作流
3. 工作流将在每天北京时间9:00自动运行

## 注意事项

1. **邮箱配置**：确保使用正确的QQ邮箱授权码
2. **时区设置**：GitHub Action使用UTC时间，已配置为北京时间9:00
3. **资讯来源**：当前为模拟数据，实际使用时需要接入真实的财经API
4. **风险控制**：所有投资建议均包含风险提示

## 自定义配置

编辑 `config.json` 文件可以调整：
- 邮件接收地址
- 发送时间
- 资讯数量和来源
- 邮件模板

## 故障排查

1. **邮件发送失败**：检查邮箱授权码是否正确
2. **工作流不运行**：确认GitHub Action已启用
3. **时间不准确**：检查工作流中的cron表达式
